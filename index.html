Comandos de Terminal

Para ver las opciones de cada uno de los comandos de la terminal podemos agregar --help

  • ls Lista el contenido de una carpeta
  • cd Moverse entre carpetas
  • code Abre el editor de código
  • start Abre la carpeta
  • touch Crea un archivo vacio
  • echo Crea contenido en un archivo
  • cat Permite ver la información de un archivo
  • nano Permite abrir un mini editor en la terminal
  • rm Permite eliminar un archivo
  • clear Limpia la terminal
  • Mkdir Crea otra carpeta
  • Find Permite encontrar archivos en un determinado directorio a partir de diversas reglas de búsqueda
